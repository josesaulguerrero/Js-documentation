Introduction to JavaScript

JavaScript is a scripting Object-oriented language which allows us to give interactivity to our websites. When you see cool animations, updated content, interactive elements, search bars and much more, JavaScript is likely involved.

It gives us the ability to save values, perform numerical operations, run a piece of code (known as script) when an element is manipulated, make amazing animations, change our HTML, handle information coming from an API (Application Programming Interfaces).

Briefly, JavaScript is the one which give us the power to make our websites greater with simple and short pieces of code.

JavaScript's special features

JavaScript has certain features that differentiate it from the rest, a quick example, when you write:

var sum = '12' + 5;

The value of "sum" will be '125'. Weird, right ?

let's go deeper on these special features

  • Weak-typed:

    In JavaScript, the data types are not defined, therefore, there is no difference between declaring a text string or a numerical variable.

    var helloWorld = 'hello world';
    var seventeen = 17;

    Also, there is no need to use the semicolon as we would in other languages. We could simply write:

    var helloWorld = 'hello world'
    var seventeen = 17

    And it would work without giving us any errors !

  • Interpreted:

    Unlike other languages, JavaScript does not need to be compiled beforehand, you can paste your script on your favorite browser's console and that's all, the browser will do the work and you will see how the code runs in real time.

    As you saw, you can code and test your scripts on your favorite browser.

  • Backwards:

    This is an interesting feature of this language, why? Because we can use both the old and new syntax and the code will run normally, this is due to its backwards ability that gives us the power to use the old syntax without worry.

    'var' is an old way of declaring variables and 'let' is the newer way (we will cover that in the following topics) and as we see, the code run as usual, no errors were logged on the console.

Data types in JavaScript

This language counts with two data types, primitive and non-primitive data values, also known as objects. let's dive into these data types:

Primitive data values

  • Number:

    These are just numerical data values.

    var OneTwoThree = 123;

  • String:

    It is text written within quotation marks.

    var hello = "hello world";
    var anotherString = 'hello world'

  • Boolean:

    This data value is really useful, because it tells us if something is true or false.

    var aBoolean = true;
    var anotherBoolean = false;

  • Empty values:

    These data values are a bit confusing for some people, but here you will completely learn the difference between them. Within this category we find Undefined and Null values; undefined variables are those one which have never been declared or created and null variables refer to the ones which have been declared without any value.

Non-primitive data values

  • Arrays:

    It's a linear data set which allows us to save many values into one variable, like this:

    Var array = [1,2,3,4];
    var arrayWithStrings = ['a', 'b', 'c'];

    to access its values we put:

    array[anyPosition]

    the positons start from 0, therefore array[1] is equal to 2.

  • Objects:

    These data types are really important in object-oriented programming, because we can save values here, with a respective key:

    var dog = {
       name: 'Bailey',
       age: 17
    }

    and to access these values we write:

    dog.name

Variables in JavaScript

As we showed you in previous topics, there are different variable types in JavaScript. Let's go further in this topic:

  • Var:

    It's an old mean to declare variables, but it had some problems and that is why there are another kind of variables.

    var varVariable = 123;

  • Let:

    This way to declare variables is newer ann it has solved many of the problems caused by 'var', it works quite similarly though.

    let letVariable = 123;

  • Const:

    Once we declare a variable with this keyword, we can not change its value with any method, it will have an eternal value.

    const constVariable = 123;

Commenting out

The comments are something really useful, due to their ability to "hide" code, when we comment out a piece of code, this piece won't be ran by the browser, altough we can see it and it is still in the script. There are two kinds of comments in JavaScript, let's check them out:

  • Single-line comments:

    The single-line comments are those which give us the chance to comment out only one line, no more. We write them like this:

    // let number = 123;

    As shown above, to write a single-line comment, we put a pair of slashes before the code line we wanna comment out.

  • Multiline comments:

    These ones can comment a whole script without any problem and we can skip this whole piece of code, because the browser will ignore it. This is their syntax:

    /* let number = 123;
    console.log(number)*/

    The way to create them is putting the code piece we wanna skip within /* */.

Conditionals

All programming languages count with if statement, and that is why we can define if something is true or false. Let's read more detailed information on it:

If statement

  • This statement gets a certain condition and based on it, it runs a piece of code or another, that is the reason why it is so useful, because we can run a certain code or simply skip it if the condition is false. The sintax is:

    if(condition) {
       // the code we want to run if the condition is true
    }

  • The switch statement is used to perform different actions based on different conditions.The structure is:

    switch(expression) {
      case x:
        // code block
        break;
      case y:
        // code block
        break;
      default:
        // code block
    }

    This is how it works:

    • The switch expression is evaluated once.
    • The value of the expression is compared with the values of each case.
    • If there is a match, the associated block of code is executed.
    • If there is no match, the default code block is executed.

Loops in JavaScript

The loops are control structures that run a piece of code once or several times as specified. This is useful when we need to go through an array or do something while a codition is true. There are two sorts of loops, while loops and for loops, let's give them a look.

  • For loop:

    This loop runs a piece of code a certain number of times, we must define this number beforehand, otherwise the loop won't work. The syntax is:

    for(iterationVariable; numberOfTimes; increment) {
       // piece of code
    }

    it should be clarified that the semicolons within the parethesis are necessary.

  • While loop:

    With this control structure we can execute some code as long as the condition is true, besides, we do not have to define how many times we want the loop to run. Its structure is:

    while(condition){
      // code to be ran
    }
